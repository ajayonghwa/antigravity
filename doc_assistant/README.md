# 오프라인 문서 작성 도우미 (Doc Assistant)

이 툴은 오프라인 환경에서 로컬 LLM(Ollama)을 활용하여 전문적인 문서를 작성하고, 이미지를 자동으로 정리해주는 파이썬 프로그램입니다.

## 🛠 준비 사항

1.  **Python 설치**: Python 3.8 이상이 설치되어 있어야 합니다.
2.  **Ollama 실행**: Ollama가 설치되어 있고 모델이 다운로드되어 있어야 합니다. (기본: `gemma4:e4b`)
3.  **가상환경 설정 및 라이브러리 설치**:
    - 본 프로젝트는 `doc_assistant/` 폴더 내의 독립된 가상환경을 사용합니다.
    - **Windows**:
      ```powershell
      .\.venv\Scripts\activate
      pip install requests python-docx pymupdf
      ```
    - **Mac/Linux**:
      ```bash
      source .venv/bin/activate
      pip install requests python-docx pymupdf
      ```

## 📁 폴더 구조

- `references/`: 참고할 PDF 또는 TXT 문서를 넣어주세요. (암호화된 PDF 지원)
- `images/`: 주제별 폴더를 만들어 이미지를 넣어주세요. (예: `images/결과분석/`)
- `content/`: 작업 중간 단계의 데이터가 저장됩니다.
- `output/`: 최종 생성된 `.docx` 파일이 저장됩니다.

## 🚀 사용 방법

1.  **Step 1: 개요 생성**
    *   `references/`에 참고 자료를 넣습니다.
    *   `python main.py` 실행 후 `1`번을 선택합니다.
    *   주제를 입력하면 LLM이 개요를 작성합니다.
2.  **Step 2: 대화형 내용 작성 (중단/재개 지원)**
    *   `python main.py` 실행 후 `2`번을 선택합니다.
    *   **중단**: 작업 중 `q` 또는 `exit`를 입력하면 현재까지 내용을 저장하고 종료합니다.
    *   **재개**: 다시 실행 시 기존 진행 상황을 요약해 보여주며, 이어서 하기/처음부터 하기/특정 섹션부터 하기 옵션을 제공합니다.
    *   각 섹션에 대해 구어체로 설명하면 LLM이 전문적인 문체로 다듬어 저장합니다.
3.  **Step 3: 최종 문서 생성**
    *   `images/` 폴더 내에 섹션 이름과 매칭되는 폴더를 만들고 이미지를 넣습니다.
    *   `python main.py` 실행 후 `3`번을 선택합니다.
    *   `output/` 폴더에 이미지가 포함된 `.docx` 문서가 생성됩니다.
4.  **Step 4: 개요 및 내용 수동 수정**
    *   AI가 생성한 초안이 마음에 들지 않거나, 새로운 섹션을 직접 추가하고 싶을 때 사용합니다.
    *   기존 섹션의 내용을 직접 타이핑하여 수정하거나, 새로운 섹션을 즉시 추가할 수 있습니다.

## ⚠️ 참고 사항 (윈도우 11 및 보안 PDF)

- **보안 PDF**: 표준 암호가 걸린 경우 `config.json`의 `password` 필드에 암호를 입력하면 읽을 수 있습니다. 기업용 보안 솔루션(DRM)이 적용된 경우 수동으로 해제 후 넣어주셔야 합니다.
- **HWP 변환**: 생성된 `.docx` 파일은 아래한글(HWP)에서 직접 열거나 '불러오기'를 통해 완벽하게 호환됩니다.
- **Windows 호환**: 본 스크립트는 윈도우 11의 경로 체계(Backslash)를 자동으로 지원하도록 설계되었습니다.
